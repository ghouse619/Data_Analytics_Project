## How to Run Any Project

```bash

pip install pandas numpy matplotlib seaborn scikit-learn scipy

# Run any project
python 01_ecommerce_analysis/ecommerce_analysis.py
python 02_food_delivery_forecasting/food_delivery_analysis.py
python 03_student_dropout_prediction/student_dropout_analysis.py
python 04_hr_attrition_analysis/hr_attrition_analysis.py
```

Each script generates:
- **Cleaned dataset** → `data/`
- **5 analysis charts (PNG)** → `charts/`
- **CSV outputs** (segments, scores, rankings) → `outputs/`
- **Written report** with insights & recommendations → `outputs/analysis_report.txt`
